# 🍩 Proyecto Fábrica de Donuts

Este proyecto contiene simulaciones de fábricas de donuts usando **hilos (threads)** y **semáforos** para controlar el acceso a recursos compartidos.

---

## Paquetes y descripción

### 1. `cdi.teoria.SemaforoConteo`
Simula una fábrica de donuts donde los clientes compiten por entrar a la zona de degustación usando semáforos para controlar el acceso. 🍩🏃‍♂️

### 2. `cdi.teoria.SemaforoBinario`
Simula una fábrica de donuts donde los trabajadores usan un semáforo binario para turnarse en la máquina y preparar los donuts de manera segura. 🍩🔧

### 3. `cdi.teoria.SemaforosCombinados`
Simula una fábrica de donuts donde los clientes compiten por entrar a la zona de degustación y además deben turnarse para usar el baño, usando semáforos combinados para controlar ambos recursos. 🍩🚽🏃‍♂️

### 4. `cdi.teoria.SemaforoBinarioMaquinas`
Simula una fábrica de donuts donde los trabajadores comparten varias máquinas, usando un semáforo binario para turnarse y fabricar donuts de manera ordenada y segura. 🍩🔧😎
